# Як додати свій пак

Якщо маєш ідею для паку — кидай PR. Без бюрократії, без код-рев'ю на 40 коментів.

## Що потрібно

1. **JSON-файл** у `packs/твій-пак.json`:

```json
{
  "spinnerVerbs": {
    "mode": "replace",
    "verbs": ["фраза раз", "фраза два", "фраза три"]
  }
}
```

2. **Slash-команда** у `.claude/commands/install-spinner-твій-пак.md`:

```
Встанови [назва] пак спіннерів.

1. Прочитай файл `packs/твій-пак.json` з поточної директорії
2. Відкрий або створи файл `~/.claude/settings.json`
3. Якщо файл існує — зроби merge: додай або заміни ключ `spinnerVerbs` зберігши всі інші налаштування
4. Якщо файлу немає — створи з вмістом з packs/твій-пак.json
5. Повідом що встановлено успішно і як виглядатиме спіннер
```

3. **README.md** — додай свій пак у таблицю і блок з прикладами.

4. **CLAUDE.md** — додай рядок у список паків і slash-команд.

## Правила (їх мало)

- Пиши українською. Суржик, сленг, мемна мова — ок. Головне щоб було зрозуміло.
- Фраз має бути хоча б 20. Менше — буде повторюватись і набридне.
- Без образ, без політичних срачів, без токсику. Гумор — так, хейт — ні.
- Фрази мають пасувати до контексту лоадера — це те що бачиш поки Claude думає. Тобто щось про процес, роздуми, очікування.
- В кінці кожної фрази не треба ставити `...` — Claude Code додає їх сам.

## Ідеї для паків

Не знаєш що зробити? Ось кілька напрямків:

- **PM** — дедлайн був вчора, а що по статусу?
- **Дизайнер** — піксель лівіше, а тепер правіше
- **QA** — а якщо натиснути двічі? а тричі?

Або вигадай своє. Якщо смішно тобі — скоріш за все смішно і іншим.

## Як пушити

Форкни, зроби зміни, кинь PR. В описі напиши що за пак і для кого. Все.
